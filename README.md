# Исследование надёжности заёмщиков

## Постановка задачи

Заказчик — кредитный отдел банка. Цель исследования — выяснить, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок. Результаты будут использованы при построении модели **кредитного скоринга** — системы оценки способности заёмщика вернуть кредит.

## Данные

Файл `data.csv` содержит статистику о платёжеспособности клиентов банка:

| Столбец | Описание |
|---------|----------|
| `children` | Количество детей в семье |
| `days_employed` | Общий трудовой стаж в днях |
| `dob_years` | Возраст клиента |
| `education` | Уровень образования |
| `family_status` | Семейное положение |
| `gender` | Пол клиента |
| `income_type` | Тип занятости |
| `debt` | Наличие задолженности (целевой признак) |
| `total_income` | Ежемесячный доход |
| `purpose` | Цель получения кредита |

## Этапы работы

### 1. Предобработка данных
- Заполнение пропусков в `total_income` и `days_employed` медианными значениями по типу занятости
- Замена отрицательных значений трудового стажа (`abs()`)
- Удаление аномалий в столбце `children` (значения -1 и 20)
- Приведение столбца `education` к нижнему регистру
- Удаление 71 дубликата
- Категоризация доходов (`total_income_category`: A, B, C, D, E)
- Категоризация целей кредита (`purpose_category`)

### 2. Исследовательский анализ

**Количество детей** — небольшая зависимость: у заёмщиков с 1–2 детьми процент должников выше (9.23–9.45%), чем у бездетных (7.54%).

**Семейное положение** — есть зависимость: незамужние/неженатые (9.76%) и состоящие в гражданском браке (9.31%) чаще допускают просрочку, чем вдовцы (6.62%) и разведённые (7.06%).

**Уровень дохода** — заёмщики категории C (50k–200k) имеют 8.50% просрочек, категория B (200k–1M) — 7.06%.

**Цель кредита** — операции с автомобилем (9.35%) и получение образования (9.25%) дают наибольший процент просрочек, недвижимость (7.26%) — наименьший.
